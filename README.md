
# About

iOS用の細かい機能の集合ライブラリです



# 機能一覧

* CustomFirstSplash
    * unityが出力したxcodeプロジェクトのsplashを上書きし、ロゴ画像を表示できます

* noBitCode
    * ビルド時にBitCodeを自動でオフにする機能


* InfoPlistConfig
    * ビルド時に、InfoPlistへの権限などの設定を自動で行う機能


* Cameralight
  * 端末のライトのオン、オフができます
 
    
* Keychaingラッパー
  * keychainのシンプルな機能をつかうことができます
    
* Vibrate
  * 端末の振動機能をつかうことができます
